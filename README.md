### ⚙️ LOW-LEVEL STACK IMPLEMENTATIONS

I HAVE INDEPENDENTLY DESIGNED AND IMPLEMENTED A COMPREHENSIVE LOW-LEVEL STACK FROM SCRATCH, INCLUDING:

- **VMM:** A VIRTUAL MEMORY MANAGER SUPPORTING 5-LEVEL PAGING, 1GB/2MB HUGE PAGES, AND COPY-ON-WRITE WITH DYNAMIC HUGE PAGE SPLITTING.
- **PMM:** A 3-LEVEL PHYSICAL MEMORY MANAGER FEATURING LAZY BITMAP INITIALIZATION AND PER-CPU CACHES TO ELIMINATE GLOBAL SPINLOCK CONTENTION.
- **ALLOCATOR:** AN INDUSTRIAL-GRADE CONCURRENT `MALLOC`/`FREE` ENGINE UTILIZING QSBR (QUIESCENT-STATE-BASED RECLAMATION) GARBAGE COLLECTION, TLS BATCHING, AND LOCK-FREE BITMAP CAS OPERATIONS.
- **VFS & FD:** A VIRTUAL FILE SYSTEM WITH HASHMAP MOUNT POINT RESOLUTION AND A BLAZING-FAST BITMAP-BASED FILE DESCRIPTOR ALLOCATOR WITH O(1) TAIL INSERTION.
- **GRAPHICS:** A TTF RASTERIZATION ENGINE WITH LRU+HASHTABLE GLYPH CACHING, CJK TYPOGRAPHY RULES, AND STRICT BOUNDARY-CLIPPED ALPHA BLENDING FOR THE LINEAR FRAMEBUFFER.
